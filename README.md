
---

## 🚀 Features

- Responsive navigation bar
- Hero section with call to action
- Doctor listing cards
- Appointment form with:
  - Full name
  - Email
  - Phone number
  - Doctor selection
  - Date and time picker
- Modern layout using flexbox and media queries

---

## 🛠️ Technologies Used

- ✅ HTML5
- ✅ CSS3 (Flexbox)
- ❌ No JavaScript yet
- ❌ No Backend (future scope)

---

## 📦 Setup Instructions

1. Clone or download this repository.
2. Open the folder in **VS Code** or any code editor.
3. Make sure the files are structured correctly:
   - `index.html`
   - `style.css`
4. Open `index.html` in your browser (or use **Live Server** extension).
